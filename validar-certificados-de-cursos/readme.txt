ValidateCertify is the ultimate plugin for ensuring the authenticity and integrity of issued certificates.

== Description ==

= ValidateCertify: Authenticity and trust in certificate validation =

With ValidateCertify, you ensure the reliability of your certificates and offer an efficient solution for their validation. Its configuration is quick and easy:

1. Upload your database with key information (name, surname, course, date and certificate code).
2. Allow your users to validate certificates using a shortcode or widget, without complications.

###ValidateCertify Features – Free###

- **Manual certificate management:**

> - Add new certificates individually.
> - Edit or delete existing certificates using their unique code.
> - View and manage your entire database from an intuitive interface.

- **Simplified validation:**

> - Integrate a shortcode to display the validation option on any page or post.
> - Display key certificate information: first name, last name, course, date, and code.

== Installation ==

= Minimum Requirements =

* WordPress 6.3 or greater
* PHP version 7.4 or greater
* MySQL version 5.0 or greater

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to ValidateCertify> Add New Certificate

== Frequently Asked Questions ==

**How do I install ValidateCertify**

To install the free version of ValidateCertify, follow the steps below:
From your WordPress dashboard -> Go to Plugins -> Click on 'Add new'-> In the Search field, enter ValidateCertify.
Press install -> After installation, click Activate.

**Is ValidateCertify compatible with Gutenberg?**

ValidateCertify and Gutenberg work seamlessly together. As a user, you can easily decide which editor to use at every point while editing your site.

**How to add shortcode to verify certificates**

It's as easy as using the [ValidateCertify] shortcode.

**How many fields do you have**

Have five parameters, Name, Last name, course, date and certificate code.

**Can I request customization of the fields?**

You can request customization from the ProPlus version and add the fields you need. Please contact our support team for a quote.

**Available languages**

English (United States) and Spanish.
